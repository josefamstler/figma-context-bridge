# Prompt Examples

## Implement A Component

```text
Implement this component from Figma:
https://www.figma.com/design/FILE_KEY/name?node-id=12-34
```

## Update Existing UI

```text
Update the existing component to match the latest Figma design:
https://www.figma.com/design/FILE_KEY/name?node-id=12-34
```

## Allow Adaptation Explicitly

```text
Implement this Figma component, but adapt it to our existing design system where needed:
https://www.figma.com/design/FILE_KEY/name?node-id=12-34
```

## Debug Figma Output

```text
Use the figma-design skill and show me the figma-inspect summary for this design before implementing:
https://www.figma.com/design/FILE_KEY/name?node-id=12-34
```
